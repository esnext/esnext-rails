# esnext-rails

[esnext][esnext] adapter for the Rails asset pipeline.

## Installation

Add the esnext-rails to your Gemfile:

```ruby
gem 'esnext-rails'
```

## Running tests

```
$ bundle install
$ bundle exec rake test
```

If you need to test against local gems, use Bundler's gem :path option in the Gemfile.
